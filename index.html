Git

Gitas yra populiariausia kodo versijavimo programa.

Git komandos

  • git init - sukuria gito repozitorija dabartiniame aplankale
  • git status - nurodo informacija apie lokalios gito repozitorijos statusa
  • git add - prideda nurodytus failus prie commitui paruostu failu
  • git rm -- cached - isima nurodytus failus is stageinimo vietos
  • git commit -m "zinute" - sukuria kodo versija is stageintu failu ir prideda zinute apie pakitimus (zinute trumpa ne ilgesne nei 50 simboliu)
  • git restore - grazina failus i buvusio commito busena
  • git push - is lokalios repozitorijos nusiuncia commitus i isorine repozitorija
  • git pull - is isorines repozitorijos atsiuncia commitus i lokalia repozitorija
  • git clone - klonuoja repozitorija
  • log - isveda commitu sarasa
